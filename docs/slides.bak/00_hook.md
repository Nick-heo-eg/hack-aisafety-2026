# Slide 00 — Hook (10 seconds)

> 영상의 첫 화면. 검은 배경 + 큰 글씨 한 줄. 0:00-0:10.
> 시청자가 *영상을 끝까지 볼지* 결정하는 10초.

---

## 🎬 화면

```
┌──────────────────────────────────────────────────┐
│                                                  │
│                                                  │
│        AI 에이전트는 항상                          │
│        "완료했습니다" 라고 말합니다.                │
│                                                  │
│        ┃ 어느 단계의 완료인지는,                   │
│        ┃ 아무도 모릅니다.       ┃                  │
│                                                  │
│                                                  │
└──────────────────────────────────────────────────┘
                  (검은 배경)
```

마지막 두 줄(`어느 단계의 완료인지는, 아무도 모릅니다`)은 **흰색 박스 강조 +
0.5초 늦게 등장**. *"진짜 했는지"* 가 아니라 *"어느 단계인지"* — framing 진화의
정확한 한 줄.

---

## 🎤 음성 (10초)

> "AI 에이전트는 항상 *'완료했습니다'* 라고 말합니다.
> **어느 단계의 완료인지는, 아무도 모릅니다.**"

→ 마지막 문장 *천천히* + 약간 낮은 톤.

---

## 자막 (영어, 화면 하단)

```
AI agents always say "done."
Nobody asks: done at which stage?
```

---

## 디자인

- 폰트: Sans-serif, 굵게 (Pretendard / Inter)
- 글자 크기: 화면 폭의 60%
- 배경: 순수 검정 또는 매우 어두운 그레이
- 음악: 없거나 *매우 잔잔한* drone — 음성에 집중
- 컷 전환: hook → thesis로 *fade-out → fade-in* (0.5초)

---

## ⚠️ 절대 하지 말 것

- 로고 / 팀 이름 — 영상 마지막 closing에 넣음
- 제품 이름("Honesty Verifier") — 아직 *문제*만 보여줌, 우리는 다음 컷에서
- 영어와 한국어 동시 화면 — 음성 한국어 + 자막 영어로 분리
- "이런 사례가 있습니다" — 사고 사례는 *데모가 보여줌*, hook에선 추상적으로
